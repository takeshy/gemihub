# GemiHub Markdown Preview Reference

GemiHub Markdown preview supports:

- GitHub Flavored Markdown via `remark-gfm`
- syntax highlighting for fenced code blocks
- Mermaid rendering for code fences marked `mermaid`
- callout rendering for blockquotes that start with `[!type]`
- YAML frontmatter editing through the properties panel

Markdown files have three modes:

- Preview: read-only rendered Markdown
- WYSIWYG: rich Markdown editing
- Raw: source Markdown editing

Use source-compatible Markdown. Avoid relying on custom HTML unless the user explicitly asks for it.
