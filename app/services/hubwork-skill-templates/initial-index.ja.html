Webpage Builder スキルとは

チャットでの指示から、実際に動く Web サイトを構築する組み込み AI スキルです。「ログイン付きの会員サイト」「問い合わせフォーム」「予約ページ」「ブログ」などを伝えると、HTML、API ワークフロー、認証まわり、シートスキーマまで一式を、Drive 上の web/ 配下に生成します。

  • ✓ 公開ページ / マジックリンクログイン保護ページ
  • ✓ Google Sheets を裏に持つ JSON API エンドポイント
  • ✓ 問い合わせフォーム、予約、カレンダー連携
  • ✓ キャンセル / 返信 / 無断キャンセル管理用の Admin ページ
  • ✓ Drive のメモからブログ・お知らせを公開
  • ✓ 新規シート追加時の schema 自動マイグレーション

次のステップ

  1. 1. GemiHub IDE を開き、チャットパネルで webpage-builder スキルが有効になっていることを確認します。
  2. 2. 作りたいものをアシスタントに伝えます（例：「マジックリンクログインと面談予約ページ付きのパートナーサイトを作って」）。
  3. 3. アシスタントが計画を提示 → 承認 → このページを実際のサイトに差し替えます。

準備ができたら web/index.html を差し替えてください。
