# Features

* [Chat](chat.md) - AI chat with Gemini streaming, function calling (Drive/MCP/Google Search/RAG), slash commands, file references, attachments, and encrypted history.
* [Dashboard](dashboard.md) - Grid dashboard with file/base/kanban/secret-manager/timeline/workflow/web/memo-list widgets, alignment, and interval-driven workflow auto-execution.
* [Editor](editor.md) - Editor system: WYSIWYG markdown editing, workflow editing, and per-file-type viewers.
* [Search](search.md) - Search features: local search, Drive search, RAG semantic search, and Quick Open.
* [Sync](sync.md) - Push/pull sync between the IndexedDB cache and Google Drive with MD5 change detection and conflict resolution.
* [Edit History](history.md) - Edit history: how local file edits are recorded, browsed, and restored.
